Features:
1. addition, subtraction, muliplication, division
2. support for decimal numbers
3. support for parentheses
4. support for extremly large numbers
5. error detection
6. the ability to load a text file into the display

To load a text file:
1. make sure the text file is in the same directory as the jar
2. type the name of the file into the text field
3. click load
